# Релиз 1.3.0 (build 15) — заметки для сторов

Изменения сверх версии 1.2.0.

## Тренерское приложение — FitFlow CRM

### App Store «Что нового» (полный текст, лимит ~4000 симв.)

Проведение тренировки
• Переработан экран: упражнения собраны в аккуратные блоки, карточки подходов крупнее, слева — превью упражнения.
• Убрана лишняя верхняя плашка с таймером — экран стал чище.
• Редактирование подхода — крупными кнопками −/+ (с автоповтором при удержании); галочка сохраняет значения, не отмечая выполнение.
• Быстрое добавление / удаление подхода свайпом («+1» снова работает).
• По кнопке «i» — описание техники упражнения с фото и видео.

Шаблоны и база знаний
• Шаблоны тренировок делятся на «Общие» и «Персональные» (для конкретного клиента).
• Персональный шаблон можно создать прямо из карточки клиента.
• Поиск персональных шаблонов по имени клиента.

Прогресс клиента
• У фото прогресса и замеров видно, кто добавил — тренер или клиент; фото сгруппированы по дате.
• Фото открываются на весь экран, можно удалить и загрузить сразу до 3 снимков (спереди / сбоку / сзади).

Финансы
• Понятные подсказки, если в форме дохода/расхода не заполнены поля.
• Крупнее кнопки категорий; строки платежей и действия «Изменить/Удалить» — свайпом.

Навигация и поддержка
• Новое нижнее меню-навигация в разделах.
• В обращения в поддержку можно прикреплять фото и файлы.
• «Режим разработчика» (в разделе «Помощь») с кнопкой «Сообщить о проблеме»; там же переключатель «Скрыть финансы».

Прочее
• Тренировку можно провести только по согласованному занятию (предупреждение — сразу при выборе).
• Кнопка и жест «Назад» больше не закрывают приложение.
• Множество улучшений интерфейса и стабильности.

### Google Play «Что нового» (сжато, лимит 500 симв.)

Проведение тренировки: упражнения собраны в блоки, крупные карточки подходов, кнопки −/+, свайп «+1»/удаление, техника упражнения по кнопке «i» (фото/видео).
Шаблоны: «Общие» и «Персональные» под клиента + поиск.
Прогресс: автор фото/замеров, группировка по дате, полноэкранный просмотр, до 3 фото.
Финансы: подсказки в формах, свайп «Изменить/Удалить».
Поддержка: вложения. «Назад» больше не закрывает приложение.
