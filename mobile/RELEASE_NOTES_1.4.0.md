# Релиз 1.4.0 (build 16) — заметки для сторов

Изменения сверх версии 1.3.0.

## Тренерское приложение — FitFlow CRM

### App Store «Что нового» (полный текст)

Календарь
• Новая палитра статусов: согласовано — синий, на согласовании — коралловый, проведено — салатовый, неактивные/чужие — серый; текст белый.
• В календаре внутри клиента видны все занятия тренера: занятия этого клиента цветом статуса, чужие — серым.
• Занятие, проведённое без плана, ставится по времени вашего устройства.

Проведение тренировки
• Таймер общего времени тренировки в шапке; на плавающем бейдже «идёт тренировка» — таймер.
• «+1» копирует подход с текущими значениями (после правки атрибутов).
• Раскрытие подходов — только у выбранного упражнения (одноимённые больше не раскрываются все разом).
• «Продолжить» открывает проведение тренировки, а не редактор.

Оплата и рассрочка
• Карточки пакетов разворачиваются на месте; свайп — «Изменить»/«Удалить».
• Рассрочка: чекбоксы платежей, «Внесено» зелёным и остаток в скобках; график рассрочки виден и при редактировании дохода.

История и формы
• В истории тренировок — только выполненные упражнения и длительность занятия.
• Выбор клиента в занятии — через поиск; пикер «Выбрать из базы» — с поиском и категориями.
• Список клиентов: фильтры сверху, поиск под ними.

Прочее
• Нижнее меню-навигация прячется под выдвижными шторками.
• Множество улучшений интерфейса и стабильности.

### Google Play «Что нового» (≤500 симв.)

Календарь: новая палитра (синий/коралловый/салатовый/серый); внутри клиента видны все занятия тренера (чужие — серым).
Проведение: таймер времени, «+1» с текущими цифрами, «Продолжить» открывает проведение.
Оплата: раскрывающиеся пакеты, рассрочка с чекбоксами (внесено/остаток), свайп Изм./Удал.
История: только выполненные + длительность. Поиск в форме занятия и «Выбрать из базы».
