# База знаний тренера — KnowledgeBasePage.tsx

**Маршрут:** `/knowledge` · **Точки входа:** таб-бар тренера; возврат после
сохранения/удаления упражнения или шаблона (`navigate('/knowledge', {replace})`).
**Назначение:** каталог тренера — две вкладки (Тренировки/Упражнения) с поиском,
чипами фильтров (группа мышц → подгруппа) и FAB для создания.

## Макет (сверху вниз)

1. **Заголовок** «База знаний» (32px bold, `text-ink`), отступ `px-2 pt-3`.
2. **Поиск** (`shelf`, rounded-2xl): иконка `Search` слева, `placeholder`
   «Поиск тренировок, упражнений, статей». Пока `query !== ''` — кнопка `X`
   справа (очистить).
3. **Сегмент-табы** (контейнер `bg-card-elevated`, `rounded-2xl`, `role=tablist`):
   две кнопки `flex-1` — **Тренировки** / **Упражнения**. Активная — `bg-accent
text-accent-on`, неактивная — `text-ink-muted` (это единственный acid-fill в
   шапке). Смена вкладки сбрасывает все фильтры.
4. **Чипы основной группы** (если есть, `flex-wrap`): «Все» + список. Активный —
   `bg-accent text-accent-on`, прочие — `bg-chip text-ink-muted`, `font-mono text-xs`.
   - Вкладка Тренировки → группы мышц, реально встречающиеся в упражнениях шаблонов.
   - Вкладка Упражнения → уникальные категории упражнений каталога.
5. **Чипы подгруппы** (только если выбрана группа и подгруппы есть): «Все» + список.
6. **Список** (`flex-1 overflow-y-auto`):
   - Тренировки — строка-`shelf`: круглый бейдж `bg-chip` с числом упражнений
     (`t.exercises.length`), название (16px semibold), мета-строка `font-mono`
     «`categoryTag` · N упр.», `ChevronRight`.
   - Упражнения — `ExerciseRow` (min-h 84px): миниатюра 112px слева
     (`thumbUrl ?? imageUrl`, `object-contain`; нет картинки → плейсхолдер
     `Dumbbell`), название (line-clamp-2, 16px semibold), мета `font-mono` =
     категория + 4 иконки-параметра: повторы (`defaultReps`), вес
     (`defaultWeightKg`), время (`defaultTimeSec`), отдых (`restSec`). `ChevronRight`.
7. **FAB** (`sticky bottom-4`, справа, `tile-shadow-primary`, иконка `Plus` 24px):
   создать тренировку (на вкладке Тренировки) или упражнение.

## Данные

| Поле                        | Источник                                                                                                                       | Формат             |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------ | ------------------ |
| список упражнений           | `useExercises()` → `GET /api/exercises` → `exercises[]`                                                                        | `ExerciseResponse` |
| список шаблонов             | `useTemplates()` → `GET /api/workout-templates` → `templates[]`                                                                | `TemplateResponse` |
| чипы групп (Тренировки)     | категории упражнений, что встречаются в `t.exercises` (через карту `exerciseId→category`), порядок `GROUP_ORDER` затем алфавит | string[]           |
| чипы категорий (Упражнения) | уникальные `e.category`, алфавит (ru)                                                                                          | string[]           |
| подгруппы (Упражнения)      | `subgroupsFor(category)` — вся таксономия группы                                                                               | string[]           |
| подгруппы (Тренировки)      | подгруппы упражнений выбранной группы из шаблонов, `orderSubgroups`                                                            | string[]           |
| бейдж шаблона               | `t.exercises.length`                                                                                                           | число              |
| мета шаблона                | `t.categoryTag` + `t.exercises.length` упр.                                                                                    | строка             |
| параметры строки упр.       | `defaultReps`/`defaultWeightKg`/`defaultTimeSec` (`?? 0`), `restSec`                                                           | числа              |

`GROUP_ORDER` = Грудь, Спина, Ноги, Плечи, Руки, Корпус, Пресс/Кор, Кардио,
Растяжка, Йога (остальные категории — следом, по алфавиту).

## Действия

- Ввод в поиск → фильтрация (`rankBySearch` с нормализацией ё/е, префиксами,
  опечатками и ранжированием). Упражнения ищутся по `name`; шаблоны — по
  `name + categoryTag`. Кнопка `X` → `setQuery('')`.
- Тап по сегмент-табу → смена вкладки + сброс `category/templateGroup/subgroup`.
- Тап по чипу группы → выбор + сброс подгруппы.
- Тап по чипу подгруппы → фильтр.
- Тап по строке упражнения → `/knowledge/exercises/:id/edit`.
- Тап по строке шаблона → `/knowledge/templates/:id/edit`.
- Тап по FAB → `/knowledge/templates/new` или `/knowledge/exercises/new`.
- **Мутаций нет** (только чтение + навигация). Создание/правка — на дочерних экранах.

## Состояния

- **loading** (`exercises.isPending || templates.isPending`): «Загрузка…».
- **error** (вкладочный): «Не удалось загрузить тренировки/упражнения. Попробуйте
  обновить страницу.» (`role=alert`, нейтральный `text-ink-muted`).
- **empty каталог**: «Пока нет тренировок. Создайте первую.» / «Пока нет
  упражнений. Добавьте первое.».
- **empty фильтр** (каталог непуст, но фильтр пуст): «Ничего не нашлось.».
- Чипы группы скрыты, если список пуст; чипы подгруппы — только при выбранной группе.

## Навигация

строка упр.→`/knowledge/exercises/:id/edit` · строка шаблона→
`/knowledge/templates/:id/edit` · FAB→`/knowledge/exercises/new` |
`/knowledge/templates/new`.

## Бизнес-правила и edge-cases

- **Сохранение состояния** в `sessionStorage` (ключ `knowledge.view`): вкладка,
  query, category, templateGroup, subgroup и `scrollTop`. При возврате с дочернего
  экрана восстанавливаются вкладка/фильтры и позиция скролла (ретраи до 30 кадров,
  т.к. список тянется по высоте). Скролл-контейнер — внутренний список и/или `<main>`.
- Фильтр шаблонов по подгруппе: шаблон проходит, если есть упражнение выбранной
  группы И (подгруппа не выбрана ИЛИ у упражнения нет подгруппы ИЛИ совпадает).
- У шаблона своего поля «группа мышц» нет — она выводится через категории его
  упражнений (карты `groupByExerciseId` / `subgroupByExerciseId`).
- Миниатюра упражнения: при ошибке загрузки (`onError`) → плейсхолдер `Dumbbell`.

## Сводка эндпоинтов

- `GET /api/exercises` → `{ exercises: ExerciseResponse[] }` — каталог (личные + глобальные).
- `GET /api/workout-templates` → `{ templates: TemplateResponse[] }` — шаблоны тренера.
