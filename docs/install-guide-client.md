# Как добавить приложение «Мой тренер» на телефон

Приложение открывается прямо в браузере — устанавливать ничего из магазинов не нужно.
Достаточно один раз добавить его на главный экран, и дальше оно будет открываться
с иконки, как обычное приложение.

Адрес приложения: **https://my.fitbond.ru**

---

## 📱 Android (Google Chrome)

1. Откройте **Chrome** на телефоне.
2. В адресной строке введите **my.fitbond.ru** и перейдите.
3. Нажмите кнопку меню — три точки **⋮** в правом верхнем углу.
4. Выберите **«Добавить на главный экран»** (иногда называется **«Установить приложение»**).
5. Во всплывающем окне нажмите **«Добавить»** (название можно оставить как есть).
6. Готово — на главном экране появится иконка «Мой тренер». Открывайте приложение с неё.

> Если пункта «Добавить на главный экран» не видно — обновите страницу и попробуйте снова,
> либо проверьте, что вы именно в Chrome, а не в другом браузере.

---

## 🍎 iPhone

На iPhone тоже **рекомендуем Google Chrome** — так приложение откроется на весь экран, как
обычное. Если Chrome не установлен, подойдёт Safari, но в нём могут оставаться элементы
браузера (адресная строка/панель).

### Chrome (рекомендуем)

1. Откройте **Chrome**.
2. Введите **my.fitbond.ru** и перейдите.
3. Нажмите кнопку **«Поделиться»** вверху справа.
4. Нажмите **«Показать больше»** (стрелочка вниз).
5. Выберите **«Добавить на экран „Домой“»**.
6. Нажмите **«Добавить»** — иконка «Мой тренер» появится на экране «Домой».

### Safari (если Chrome нет)

1. Откройте **Safari**, перейдите на **my.fitbond.ru**.
2. Нажмите кнопку **«Поделиться»** внизу — квадрат со стрелкой вверх **⬆️**.
3. Выберите **«На экран „Домой“»** → **«Добавить»**.

> Для самого удобного вида (на весь экран, без панелей браузера) используйте Chrome.

---

## Что дальше

- Открывайте приложение с иконки на главном экране — как обычное приложение.
- При первом входе войдите под своим логином и паролем (или зарегистрируйтесь, если ещё нет аккаунта).
- Подключение к тренеру — по коду или QR-коду, который даст ваш тренер.
- Если приложение попросит войти заново — это нормально, просто введите данные ещё раз.
