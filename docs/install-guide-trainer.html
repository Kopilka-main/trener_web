Как добавить приложение «Тренер» на телефон

Устанавливать ничего из магазинов не нужно — добавьте на главный экран один раз.
https://app.fitbond.ru

📱 Android (Google Chrome)

  1. Откройте Chrome на телефоне.
  2. В адресной строке введите app.fitbond.ru и перейдите.
  3. Нажмите кнопку меню — три точки ⋮ в правом верхнем углу.
  4. Выберите «Добавить на главный экран» (иногда — «Установить приложение»).
  5. Нажмите «Добавить».
  6. Готово — иконка «Тренер» появится на главном экране.

🍎 iPhone

На iPhone тоже рекомендуем Google Chrome — так приложение откроется на весь экран, как обычное. Если Chrome не установлен, подойдёт Safari, но в нём могут оставаться элементы браузера (адресная строка/панель).

Chrome (рекомендуем)

  1. Откройте Chrome, перейдите на app.fitbond.ru.
  2. Нажмите кнопку «Поделиться» вверху справа.
  3. Нажмите «Показать больше» (стрелочка вниз).
  4. Выберите «Добавить на экран „Домой“» → «Добавить».

Safari (если Chrome нет)

  1. Откройте Safari, перейдите на app.fitbond.ru.
  2. Нажмите «Поделиться» внизу — квадрат со стрелкой вверх ⬆️.
  3. Выберите «На экран „Домой“» → «Добавить».
Для самого удобного вида (на весь экран, без панелей браузера) используйте Chrome.

Что дальше

  1. Открывайте приложение с иконки на главном экране — как обычное приложение.
  2. Войдите под своим логином и паролем тренера.
  3. Если приложение попросит войти заново — просто введите данные ещё раз.
